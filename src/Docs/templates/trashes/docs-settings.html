設定屬性

此模組有下列設定可供使用。欲更改已初始化的模組設定，請透過 setting 方法並傳入欲覆蓋的設定。

預設值 註釋
{{ .Key }} {{ .Type }} {{ .Default }} {{ .Description }}
silent 布林值 false 消音所有提示，甚至是錯誤訊息。
debug 布林值 true 顯示除錯訊息。
observeChanges 布林值 true 監聽 DOM 結構異動並自動重整快取。

回呼函式
