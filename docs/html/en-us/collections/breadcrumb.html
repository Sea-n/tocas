Breadcrumb
The footprint in the sand shows where you have been. Breadcrumb are used as a navigation to indicate current location.

Types

There are different types of breadcrumb.

Breadcrumb

Basic

Home
/
Shop
/
T-Shirt
<div class="ts breadcrumb">
    <a class="section">Home</a>
    <div class="divider"> / </div>
    <a class="section">Shop</a>
    <div class="divider"> / </div>
    <div class="active section">T-Shirt</div>
</div>

With other icons

You also can use other icons to be their divider.

Home Shop
T-Shirt
<div class="ts breadcrumb">
    <a class="section">Home</a>
    <i class="right angle icon divider"></i>
    <a class="section">Shop</a>
    <i class="right angle icon divider"></i>
    <div class="active section">T-Shirt</div>
</div>

Content

Breadcrumb are made from different types of content.

Divider

You can use divider for breadcrumb to show the relationship between pages.Divider can be text or icon.

Home / Registeration /
Personal Information


Home Registeration
Personal Information
<div class="ts breadcrumb">
    <a class="section">Home</a>
    <span class="divider">/</span>
    <a class="section">Registeration</a>
    <span class="divider">/</span>
    <div class="active section">Personal Information</div>
</div>

<div class="ts breadcrumb">
    <a class="section">Home</a>
    <i class="right chevron icon divider"></i>
    <a class="section">Registeration</a>
    <i class="right chevron icon divider"></i>
    <div class="active section">Personal Information</div>
</div>

Section

Breadcrumb is made by different section which is text or a link.

Home
/
Search
<div class="ts breadcrumb">
    <div class="section">Home</div>
    <div class="divider"> / </div>
    <div class="active section">Search</div>
</div>

Link

Sections in breadcrumb are available for a link or link with text.區塊可以是連結，或者帶有連結文字。

Home
/
Search：Tissue paper
<div class="ts breadcrumb">
    <a class="section">Home</a>
    <div class="divider"> / </div>
    <div class="active section">Search：<a href="#">Tissue paper</a></div>
</div>

Status

YOu can assign different status for breadcrumb.

Active

You can show a active section easily.

Products
/
Tissue paper
<div class="ts breadcrumb">
    <a class="section">Products</a>
    <div class="divider"> / </div>
    <div class="active section">Tissue paper</div>
</div>
