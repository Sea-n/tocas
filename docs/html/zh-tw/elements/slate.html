種類

一個板岩具有多個不同的種類。

板岩

一個正常的板岩。

這是一塊普通的板岩，看起來十分無聊

外觀

你可以更改板岩的位置、大小、或者形狀。

增加內距

你可以增加板岩的內距。

增加內距 這可以讓你的聚焦看板看起來不那麼壅擠。

非常內距 可以，這很內距。

超級內距 如果你還不滿足的話還有這樣增加內距地方式。

</div>

指定方向增加內距

你還可以指定增加垂直或是水平的內距。

垂直內距 僅增加垂直內距，水平內距維持預設。

水平非常內距 這會增加水平的內距，但是垂直內距保持預設。

縮減

你可以移除所有內距、或水平和垂直內距。

縮減全部 這將使的你的板岩完全沒有內距。

水平縮減 板岩的左右內距將會消失。

垂直縮減 這樣做的話上下內距將會消失。
