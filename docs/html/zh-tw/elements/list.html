    狀態

    清單能夠有不同的狀態。

    已選擇

    清單的項目能夠呈現被選擇的樣子。

    以西結
    伊波咲
    栗宮希實
    <div class="ts segmented list">
    <div class="item">以西結</div>
    <div class="[[selected]] item">伊波咲</div>
    <div class="item">栗宮希實</div>
</div>

    自訂標點

    你能夠透過 [value] 來變更清單中的標記符號。

    1. 橙希
    2. 卡莉絲
    3. 吳雨藍
      1. 亞凡芽
      2. 橙希
      3. 卡莉絲
    <ol class="ts list">
    <li [[value]]="+">橙希</li>
    <li [[value]]="+">卡莉絲</li>
    <li [[value]]="+">
        吳雨藍
        <ol>
            <li [[value]]="-"><a class="item" href="#!">亞凡芽</a></li>
            <li [[value]]="-">橙希</li>
            <li [[value]]="-">卡莉絲</li>
        </ol>
    </li>
</ol>
