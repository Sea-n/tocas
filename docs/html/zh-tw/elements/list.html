自訂標點

你能夠透過 [value] 來變更清單中的標記符號。

  1. 橙希
  2. 卡莉絲
  3. 吳雨藍
    1. 亞凡芽
    2. 橙希
    3. 卡莉絲
<ol class="ts list">
    <li value="+">橙希</li>
    <li value="+">卡莉絲</li>
    <li value="+">
        吳雨藍
        <ol>
            <li value="-"><a class="item" href="#!">亞凡芽</a></li>
            <li value="-">橙希</li>
            <li value="-">卡莉絲</li>
        </ol>
    </li>
</ol>
