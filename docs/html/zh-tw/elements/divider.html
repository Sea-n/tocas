種類

分隔線具有不同的種類。

基本

一個最基本的分隔線。

我是微笑小安安，歡迎來到小安網站。

你將會在這裡看見一個魔法師的日常生活，

沒錯，只要你能夠單身三十年，你也可以跟我一樣成為魔法師。

 
<p>我是微笑小安安，歡迎來到小安網站。</p>
<hr class="ts divider">
<p>你將會在這裡看見一個魔法師的日常生活，</p>
<p>沒錯，只要你能夠單身三十年，你也可以跟我一樣成為魔法師。</p>

段落

分隔線可以增加間距用以區分大型段落。

這裡是一個區段，然後下一行會是新世界。
哈囉！世界！ 
這裡是一個區段，然後下一行會是新世界。
<hr class="ts section divider"> 哈囉！世界！

水平分隔線

這種分隔線可以讓你在中間插入文字。

嗶嗶嗶。

我是分隔線

沒錯，你可以在分隔線中間插入一小段文字，

而且只需要一個元素。

 
<p>嗶嗶嗶。</p>
<div class="ts horizontal divider">我是分隔線</div>
<p>沒錯，你可以在分隔線中間插入一小段文字，</p>
<p>而且只需要一個元素。</p>

外觀

分隔線的外觀可以有所不同。

隱藏

不可見的分隔線，但仍帶有分隔效果。

嗶嗶嗶。

我是分隔線

你有看見中間的分隔線嗎？。

很多人看不見。

但是很多人不說。

 
<p>嗶嗶嗶。</p>
<div class="ts hidden divider">我是分隔線</div>
<p>你有看見中間的分隔線嗎？。</p>
<p>很多人看不見。</p>
<p>但是很多人不說。</p>

縮減

分隔線可以沒有間距。

這個分隔線會沒有間距，字看起來會跟分隔線黏在一起。
當然這一行也不例外。 
這個分隔線會沒有間距，字看起來會跟分隔線黏在一起。

反色

分隔線的顏色是相反色。

下面的分隔線會和一般的分隔線顏色有所不同。
