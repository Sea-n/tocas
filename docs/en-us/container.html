Introduction

The Container keeps the content at its maximum width without being overstretched, so that the content layout will not be too wide. It is also possible to place a Grid, so that the content can be concentrated in the middle of the screen. Here is the width information between the different containers.

100% 1180px 910px 720px
Style is-fluid Default is-narrow is-very-narrow
Name Fluid Container Default Container Narrow Container Very Narrow Container

Container
