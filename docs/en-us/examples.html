Examples
Example pages built with the Tocas UI, where you can view and download example source code.
NOTCIE
These examples were made for the desktops; The layout may be distorted or cropped if viewed on a mobile device.
Applications
Mostly for SPA or progressive applications.
Chatroom
Small chat room interface is implemented through dialogue bubbles and structural frames.
Dashboard
Backend administration interface with a sidebar for managing a corporate website.
Feed
Presents a timeline where users can create and view posts.
Inbox
網頁電子信箱的郵件檢視頁面並夾帶數個檔案附件。
Network Disk
A cloud drive that organizes files by card.
Blog Editor
A personal blog backend posting page that works with Simple Markdown Editor.
Websites
能夠呈現出單體、商業網站的實際範例。
Blog
以文字元素與部份特色卡片所組成的個人部落格網站。
CMS
適用於團體、企業公告的小型商業 CMS 內容管理網站前台頁面。
Crowdfunding
透過網格系統與圖片主體實現募資平台專案格局。
Galley
呈現風景旅遊照片的靜態網站。
News
有主打內容與多個章節段落的新聞資訊彙整網站。
Shopping
帶有豐富的商品資訊以及評價指示的購物網站。
Generals
較為輔助性質的工具與其他資訊頁面。
Checkout
顯示購物車項目還有結帳地址等詳細資料的表單頁面。
File List
類似 FTP 等，較為簡潔的網路硬碟目錄與檔案資訊範例。
Invoice
用於企業之間所開立的電子發票或是出貨單。
Login Form
基本的帳號與密碼登入頁面。
Settings
網站或服務後台的系統相關設定表單。
Mail
用於寄送給網站會員的郵件內容。
Payment
於商品結帳時所請求的信用卡資料頁面並附帶商品資訊。
Pricing Plans
提供給客戶數個不同價格的方案。
Registration
基本的使用者註冊頁面，並要求性別、生日、暱稱⋯⋯等資訊。
